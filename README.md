# Example Firm Legal — Website Template

A clean, responsive one-page website template designed for professional service businesses such as law firms, consultants, and advisors.
This project demonstrates modern HTML and CSS layout techniques, responsive design, and reusable component structure.

---

## Preview

Simple professional layout including:

- Sticky header navigation
- Hero section with call-to-action buttons
- About section
- Practice areas / services grid
- Trust-building section
- Contact section with form
- Footer branding
- Responsive design for mobile and desktop
- Placeholder logo and favicon

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive design principles

## Features

- Fully responsive layout

## How to Run Locally

1. Clone the repository
   git clone https://github.com/RuanFJacobs/template-portfolio-site
2. Open the project folder
3. Open `index.html` in your browser
   Or use VS Code Live Server extension.

---

## Purpose

This template was created as a reusable starting point for small professional service websites.
It demonstrates:

- clean layout structure
- maintainable CSS organization
- responsive design fundamentals
- realistic business website structure

---

## Customization

To adapt this template for a real client:

- Replace logo files in the `/logo` folder
- Update business name and contact details
- Adjust color palette if needed
- Modify practice areas/services content
- Connect the contact form to a backend or form service
